## Start the Server

```bash
# Start with defaults (port 6379, persistence enabled)
frogdb-server

# Or with a config file
frogdb-server --config frogdb.toml
```

## Connect

FrogDB speaks the Redis protocol. Use any Redis client:

```bash
redis-cli -p 6379
```

```python
import redis
r = redis.Redis(host='localhost', port=6379)
r.set('hello', 'frogdb')
print(r.get('hello'))
```

## Verify

```bash
redis-cli PING        # PONG
redis-cli INFO server # Server info including version
redis-cli DBSIZE      # Number of keys
```

All standard Redis commands work as expected. See [Redis Differences](/compatibility/redis-differences/) for the few cases where behavior diverges.

## Configuration

Generate a default config file:

```bash
frogdb-server --generate-config > frogdb.toml
```

Common options:

```toml
[server]
bind = "0.0.0.0"       # Listen on all interfaces
port = 6379
num-shards = 0          # 0 = auto-detect CPU cores

[persistence]
enabled = true
data-dir = "/var/lib/frogdb"
durability-mode = "periodic"  # async, periodic, sync
sync-interval-ms = 1000

[memory]
maxmemory = 4294967296  # 4 GB
maxmemory-policy = "allkeys-lru"
```

See [Configuration Reference](/reference/configuration/) for all options.
