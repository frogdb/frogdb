## Features

<CardGrid stagger>
	<Card title="Redis Compatible" icon="puzzle">
		Drop-in replacement for Redis with RESP3 protocol support. Use your existing Redis clients and tools.
	</Card>
	<Card title="Persistent Storage" icon="document">
		RocksDB-backed persistence with configurable durability options including AOF and periodic snapshots.
	</Card>
	<Card title="Clustering" icon="random">
		Distributed hash slots with automatic sharding, leader election, and seamless failover.
	</Card>
	<Card title="Lua Scripting" icon="seti:lua">
		Full Lua scripting support for atomic multi-key operations with `EVAL` and `EVALSHA`.
	</Card>
	<Card title="Built with Rust" icon="rocket">
		Memory-safe, high-performance implementation with async I/O and zero-copy parsing.
	</Card>
	<Card title="Observability" icon="magnifier">
		Built-in metrics, INFO commands, and client tracking for monitoring and debugging.
	</Card>
</CardGrid>

## Quick Example

```bash
# Start FrogDB
frogdb-server --port 6379

# Connect with redis-cli
redis-cli -p 6379

# Use familiar Redis commands
127.0.0.1:6379> SET greeting "Hello from FrogDB!"
OK
127.0.0.1:6379> GET greeting
"Hello from FrogDB!"
```
