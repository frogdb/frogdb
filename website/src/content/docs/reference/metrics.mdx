FrogDB exports Prometheus metrics on the HTTP endpoint (default port <ConfigDefault path="http.port" />) at `/metrics`. Enable it with `http.enabled = true` in configuration. Every table below is generated from `frogdb_telemetry::ALL_METRICS`, the same typed registry the server uses to render the endpoint, so it cannot list a metric the binary doesn't actually export.
