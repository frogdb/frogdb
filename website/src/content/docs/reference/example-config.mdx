This is a complete FrogDB configuration file with every setting at its default value. Generate an identical one yourself with `frogdb-server --generate-config`; the block below is that exact output, captured at docs-build time so it cannot drift from the flag.

## Generate it yourself

```bash
frogdb-server --generate-config > frogdb.toml
```

<Aside type="note">
`--generate-config` currently prints a hand-maintained TOML template rather than a full serialization of `Config::default()`. It is kept in sync with the config-struct field list by convention, not by the compiler, so a newly added field can land in [Configuration Reference](/reference/configuration/) before it lands here. If a field you need is missing below, check the Configuration Reference table — it is generated directly from `Config::default()` and is authoritative for the full field list.
</Aside>
