FrogDB can be installed from source using Cargo (Rust's package manager) or run via Docker.

## Prerequisites

- **Rust 1.75+** — Install via [rustup](https://rustup.rs/)
- **RocksDB dependencies** — Required for persistence (see platform-specific notes below)

## Install from Source

```bash
# Clone the repository
git clone https://github.com/frogdb/frogdb.git
cd frogdb

# Build release binary
cargo build --release

# The binary is at ./target/release/frogdb-server
```

### Platform-Specific Dependencies

#### macOS

```bash
# Install RocksDB dependencies via Homebrew
brew install rocksdb snappy lz4 zstd
```

#### Ubuntu/Debian

```bash
# Install build dependencies
sudo apt-get update
sudo apt-get install -y build-essential librocksdb-dev libsnappy-dev liblz4-dev libzstd-dev
```

#### Arch Linux

```bash
sudo pacman -S rocksdb snappy lz4 zstd
```

## Docker

Run FrogDB without installing dependencies:

```bash
# Pull and run the latest image
docker run -p 6379:6379 frogdb/frogdb:latest

# With persistent storage
docker run -p 6379:6379 -v frogdb-data:/data frogdb/frogdb:latest
```

### Docker Compose

```yaml
version: '3.8'
services:
  frogdb:
    image: frogdb/frogdb:latest
    ports:
      - "6379:6379"
    volumes:
      - frogdb-data:/data
    command: ["frogdb-server", "--data-dir", "/data"]

volumes:
  frogdb-data:
```

## Verify Installation

After installation, verify FrogDB is working:

```bash
# Start the server
./target/release/frogdb-server

# In another terminal, connect with redis-cli
redis-cli ping
# Returns: PONG
```
