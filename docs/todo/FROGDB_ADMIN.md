# frogdb-admin

Admin diagnostic tool at `frogdb-server/ops/frogdb-admin/`.

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `generate` | Generate a diagnostic bundle |
| `list` | List available diagnostic bundles |
| `download` | Download a diagnostic bundle |

## Status

Needs user-facing documentation. The tool exists but is undocumented.
