# FrogDB Licensing

FrogDB is available under a tri-license model. You may choose whichever license best suits your needs:

## License Options

### 1. Business Source License 1.1 (BSL)

**Best for:** Most users who want to use FrogDB in their applications.

The BSL allows you to use, copy, modify, and redistribute FrogDB with one restriction: you may not use it to offer a competing in-memory database product or managed database service.

After **2 years** from each version's release date (following version 1.0.0), that version automatically converts to the permissive **Apache License 2.0**.

📄 **Full text:** [licenses/BSL-1.1.md](licenses/BSL-1.1.md)

### 2. GNU Affero General Public License v3.0 (AGPLv3)

**Best for:** Users who need an OSI-approved open-source license and are comfortable with copyleft requirements.

The AGPLv3 allows you to use FrogDB freely, including for commercial purposes and managed services, provided that:

- Any modifications you make must be released under AGPLv3
- If you offer FrogDB as a network service, you must provide source code to users

This is a strong copyleft license with no time-based restrictions.

📄 **Full text:** [licenses/AGPL-3.0.md](licenses/AGPL-3.0.md)

### 3. Commercial License

**Best for:** Organizations that want to offer FrogDB as a managed service without AGPLv3 copyleft obligations, or that have policies preventing use of BSL/AGPLv3.

Commercial licenses are available for:

- Managed database service providers (DBaaS)
- OEM/embedded use cases
- Organizations requiring proprietary modifications
- Enterprise support and indemnification

📄 **Details:** [licenses/COMMERCIAL.md](licenses/COMMERCIAL.md)

## Quick Reference

| Use Case                                                | Recommended License |
| ------------------------------------------------------- | ------------------- |
| Internal application using FrogDB                       | BSL ✓ or AGPLv3 ✓   |
| SaaS product powered by FrogDB (not a database service) | BSL ✓ or AGPLv3 ✓   |
| Distributing FrogDB with modifications (copyleft OK)    | AGPLv3 ✓            |
| Offering managed FrogDB service (copyleft OK)           | AGPLv3 ✓            |
| Offering managed FrogDB service (no copyleft)           | Commercial License  |
| Building a competing database product                   | Commercial License  |
| Government/enterprise requiring OSI license             | AGPLv3 ✓            |
