# FrogDB Release History and License Change Dates

This document tracks release dates and the corresponding BSL Change Dates for each version of
FrogDB.

Under the Business Source License 1.1, each version of FrogDB automatically converts to the Apache
License 2.0 two (2) years after its initial release date.

## Version Status Legend

| Status         | Meaning                                            |
| -------------- | -------------------------------------------------- |
| **Planned**    | Release is planned but not yet available           |
| **Current**    | Latest stable release under BSL                    |
| **BSL**        | Available under Business Source License 1.1        |
| **Apache 2.0** | BSL period expired; now available under Apache 2.0 |

---

## Notes

1. **Change Date Calculation:** The Change Date is exactly 2 years (730 days) from the Release
   Date.

2. **Timezone:** All dates are in UTC.

3. **License Selection:** Regardless of the BSL Change Date status, FrogDB is always available
   under:

   - Business Source License 1.1 (with restrictions until Change Date)
   - GNU Affero General Public License v3.0 (AGPLv3)
   - Commercial License (contact nathan.m.jordan@gmail.com)

4. **Retroactive Changes:** Once a version's Change Date has passed, that specific version is
   permanently available under Apache 2.0. This cannot be revoked.

---

## Example

If FrogDB 1.0.0 is released on **January 15, 2026**, then:

- From **January 15, 2026** to **January 15, 2028**: Version 1.0.0 is available under BSL 1.1 (or
  AGPLv3/Commercial)
- From **January 15, 2028** onward: Version 1.0.0 is also available under Apache 2.0
